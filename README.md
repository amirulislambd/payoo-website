**Welcome to Payoo** — A clean, modern, and fully responsive digital wallet UI built as a beginner frontend project.

## 📋 Project Overview

**Payoo** is a static frontend clone of a mobile financial service (MFS) app, inspired by real Bangladeshi digital wallets like bKash, Nagad, and Payoo.  

It simulates a complete wallet dashboard where users can:
- View their balance
- Perform quick transactions (Add Money, Cashout, Transfer, Pay Bill, etc.)
- Add money from bank/bKash
- See transaction history

This was one of my early beginner projects to practice **HTML, CSS, Tailwind CSS, and JavaScript** for building real-world fintech interfaces.

## ✨ Key Features

- **Modern Dashboard UI**
  - Live balance display ($57,000 example)
  - Welcome message with user info

- **Quick Action Grid**
  - Add Money
  - Cashout
  - Transfer Money
  - Get Bonus
  - Pay Bill
  - Transaction History

- **Add Money Form**
  - Select Bank (bKash example)
  - Bank Account Number input
  - Amount to Add
  - Pin Number
  - Fully styled "Add Money" button

- **Transaction History Section**
  - Shows recent transactions with date, amount, and status

- **Fully Responsive**
  - Designed mobile-first (looks perfect on phone screens)

- **Clean & Professional Design**
  - Tailwind CSS for fast and beautiful styling
  - Smooth hover effects and modern cards
